      latencyChart: 'r => ({ type: "bar", data: { labels: r.services.map(service => service.name), datasets: [{ label: "P95 latency (ms)", data: r.services.map(service => service.latency), backgroundColor: ["#00d1b2", "#3273dc", "#ffdd57", "#ff6685"] }] }, options: { scales: { y: { beginAtZero: true } } } })'

## Data contract being exercised

- `$src` loads `data/operations.json` relative to this document.
- `$derive` creates reusable table rows, filtered incidents, and a Chart.js config.
- `from` injects each resolved value into the receiving component before rendering.
