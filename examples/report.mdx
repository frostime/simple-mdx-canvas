# simple-mdx-canvas Example Report

<Callout type="summary" title="Core idea">
`simple-mdx-canvas` turns an agent-written Markdown document into a Bulma-styled HTML canvas without requiring a full frontend project.
</Callout>

<Tags>
<Tag color="primary" light>MDX</Tag>
<Tag color="success" light>Bulma</Tag>
<Tag color="info" light>Single HTML</Tag>
</Tags>

## Why this exists

Plain Markdown is portable, but it is weak at visual comparison, guided reading, and report-style presentation. A full React app is too heavy for a one-off agent artifact.

<Columns cols={2}>
<Column title="Plain Markdown">

- Low cost
- Easy to diff
- Weak visual structure
- No controlled components

</Column>
<Column title="MDX Canvas">

- Still text-first
- Adds visual components
- Generates direct HTML
- Agent can validate and repair

</Column>
</Columns>

## Recommended workflow

<Steps>
<Step title="Write the document">
The agent writes one `.mdx` file using GFM Markdown and registered Bulma-based components.
</Step>
<Step title="Validate">
The agent runs `smc validate report.mdx` and fixes any structured errors.
</Step>
<Step title="Render">
The agent runs `smc render report.mdx -o report.html`.
</Step>
</Steps>
